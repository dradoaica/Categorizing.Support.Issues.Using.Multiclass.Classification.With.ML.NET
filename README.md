# GitHubIssueClassification

| ML.NET version | API type    | Status     | App Type    | Data sources                | Scenario              | ML Task                    | Algorithms                                             |
|----------------|-------------|------------|-------------|-----------------------------|-----------------------|----------------------------|--------------------------------------------------------|
| v5.0.0         | Dynamic API | Up-to-date | Console app | .tsv file and GitHub issues | Issues classification | Multi-class classification | SDCA multi-class classifier, AveragedPerceptronTrainer |

This is a simple prototype application to demonstrate how to use [ML.NET](https://www.nuget.org/packages/Microsoft.ML/)
APIs. The main focus is on creating, training, and using ML (Machine Learning) model that is implemented in Predictor.cs
class.
